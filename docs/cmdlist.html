NAME

cmdlist - Display commands necessary to execute regular updates.

SYNOPSIS

cmdlist.sh -h

cmdlist.sh –help

cmdlist.sh [-e|–execute]

DESCRIPTION

Display commands necessary to execute regular updates. Optionally, execute them sequentially.

Optional Arguments:

-h Print command usage guide and exit

–help Print this detailed message and exit

-e|–execute Execute all commands sequentially

Execution shall fail in most cases.
