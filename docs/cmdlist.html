NAME

cmdlist - Display commands necessary to execute regular updates.

SYNOPSIS

cmdlist -h

cmdlist –help

cmdlist [-e|–execute]

DESCRIPTION

Display commands necessary to execute regular updates. Optionally, execute them sequentially.

Optional Arguments:

-h Print command usage guide and exit

–help Print this detailed message and exit

-e|–execute Execute all commands sequentially

Execution shall fail if '$'s before variables are not appropriately skipped in expressions.

See arbitrary_named_var

VARIABLES

all_cmds:

List (string) of reference to all variables, whose values shall be displayed. Separator: ' '

arbitrary_named_var:

Arbitrary expression stored and referred to in all_cmds. Variable references in expressions should be suitably skipped to defer evaluation at runtime.

EXAMPLE:

# THIS IS ILL-DISPLAYED IN MANPAGE
cmdlist_dnf_cmd="sudo su -l -c \"all_proxy=\${all_proxy} dnf --refresh -y update\""
