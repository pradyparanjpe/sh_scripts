NAME

sync_all_gitlab_issues - Synchronize issues for all gitlab projects.

SYNOPSIS

sync_all_gitlab_issues [-h]

sync_all_gitlab_issues [–help]

sync_all_gitlab_issues [[*Optional Arguments] …] CONFIG_YAML

DESCRIPTION

Synchronize issues for all gitlab projects.

A multi-wrapper for https://pypi.org/project/gitlab-issues-sync/

Optional Arguments

-h print usage message and exit

–help print this help message and exit

-f|–pull Copy from remote to local in config file (default: push)

-v|–verbose Print information inferred from configuration file

-V|–very-verbose Print secrets in information (assumes -v)

-b BLOCK|–block BLOCK Sync BLOCK from config file {default: 'default'}

Positional Arguments

CONFIG_YAML Yaml file with configuration as described below

FILES

Configuration file (yaml format) should contain following structure. Yaml indent: 2 spaces. 

BLOCK:
  local:
    url: https://gitlab.com
    token: <secret-token>
    projects: 11 12 13
  remote:
    url: https://www.example.com
    token: <secret-token>
    projects: 101 102 103
  # comment
  # inline comments WILL pollute value
  # yaml indentation: 2 spaces
