NAME

get_pass - update password_store, fetch password.

SYNOPSIS

get_pass -h

get_pass –help

get_pass [Optional Arguments*] [INSTANCE]

DESCRIPTION

Update password store git repository, then, fetch password from password-store. If instance is not provided, list available instances.

Optional Arguments

-h print usage message and exit

–help print this help message and exit

–<passargs> all passargs are passed to pass.

Optional Positional Argument:

INSTANCE fetch password from password store for INSTANCE
