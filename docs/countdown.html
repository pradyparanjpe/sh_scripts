NAME

countdown - Display a countdown timer on STDOUT, that updates every second.

SYNOPSIS

countdown -h

countdown –help

countdown [-p|progress] [-n|–notify] [-u N|–update N] [-t|–time] PERIOD

DESCRIPTION

Display a countdown timer on STDOUT, that updates every second

Optional Arguments

-h Print command usage guide and exit

–help Print this detailed message and exit

-b|–bland Print bland (without colours) output

-n|–notify Send desktop notification

-p|–progress Show progress bar

-t|–time Treat the positional argument as target time

-u N|–update N Update every N seconds

Positional Argument

PERIOD Period in seconds to count down.
