NAME

gen_self_ssl_cert.sh Generate SSL certificate key-pairs

SYNOPSIS

gen_self_ssl_cert.sh -h

gen_self_ssl_cert.sh –help

gen_self_ssl_cert.sh [Optional Arguments …] -t|–template [SSL_ROOT]

gen_self_ssl_cert.sh [Optional Arguments …] SSL_ROOT

DESCRIPTION

Generate SSL certificate key-pairs with opinionated preset options.

Optional Arguments

-h Print usage message and exit

–help Print this help message and exit

-t|–template Write configuration file template and exit.

-k KEY|–key KEY Key base [default: ${HOSTNAME} ]

-c CONF|–conf CONF Configuration file path [default: SSL_ROOT/KEY.conf ]

-y YEARS|–years YEARS Validity in years [default: 10]

Optional Positional Argument

SSL_ROOT Root for keys [ default: ${PWD} ]
