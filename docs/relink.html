NAME

relink - (Soft) Re-link all link-files

SYNOPSIS

relink -h

relink –help

relink [-v|–verbose] [-c|–commit] LINK_DIR MOD_PAT [NEW_STR]

DESCRIPTION

Re-link (soft) all link-files in LINK_DIR by replacing MOD_PAT with NEW_STR. The default behaviour is to dry-simulate the process and show a verbose output of proposed changes. If NEW_STR is not supplied (safe), all prospective affected files are shown. Changes will be committed (written) only if NEW_STR and -c flag are supplied. Blank NEW_STR may be supplied as "".

Positional Arguments:

LINK_DIR: Directory in which links are located

MOD_PAT: Pattern in link-targets to be altered

NEW_STR: Replace MOD_PAT by NEW_STR if given

Optional Arguments:

-h Display usage message and exit

–help Display this help message and exit

-m|–mock Do not perform action, only mock (assumes verbose)

-v|–verbose Display verbose outputs
