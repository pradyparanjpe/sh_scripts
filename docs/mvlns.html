NAME

mvlns - Move OBJECT from SOURCE to DESTIN and leave a soft link at SOURCE

SYNOPSIS

mvlns -h

mvlns –help

mvlns <optional arguments> SOURCE OBJECT DESTIN

mvlns <optional arguments> DESTIN

DESCRIPTION

Move OBJECT from SOURCE to DESTIN and leave a soft link at SOURCE

Optional Arguments:

-h print usage message and exit

–help print this message and exit

-p|–parents make parent paths if they don't exist

-v|–verbose print verbose

Optional Positional Argument:

DESTIN future parent path of object [default: ${PWD} ]

SOURCE future link parent [default: guessed from OBJECT]

Required Positional Argument:

OBJECT name of object to move and link {<file>|<directory>|…}
