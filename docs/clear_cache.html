NAME

clear_cache - clear cache by passing '3' to /proc/sys/vm/drop_caches

SYNOPSIS

clear_cache -h

clear_cache –help

clear_cache [-y|–assumeyes] [-n|–assumeno]

DESCRIPTION

Clear cache by passing '3' to /proc/sys/vm/drop_caches After confirmation, super-user privileges will be requested.

Optional Arguments:

-h Print command usage guide and exit

–help Print this detailed message and exit

-y|–assumeyes don't prompt, assume yes to all

-n|–assumeno don't prompt, assume no to all

FILES

Linux system file /proc/sys/vm/drop_caches.
