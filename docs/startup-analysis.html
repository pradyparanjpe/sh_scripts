NAME

startup-analysis - systemd-analyze after each startup.

SYNOPSIS

startup_analysis

startup_analysis -h

DESCRIPTION

Store systemd-analyze plot to XDG_STATE_HOME.

ENVIRONMENT

XDG_STATE_HOME

Parent directory to store image. XDG_STATE_HOME is should to be ${HOME}/.local/state.
